# 🎂 Age Calculator

A responsive Age Calculator built with that calculates a person's exact age in years, months, and days based on their date of birth.

![Age Calculator Screenshot](./screenshot_age_calculator.png)

## ✨ Features

- Calculate exact age in years, months, and days
- Birthday detection with a special greeting 🎉
- Input validation
- Error handling for invalid dates
- Reset functionality
- Responsive layout
- Clean and modern UI


## 📚 Concepts Practiced

- DOM Manipulation
- Event Listeners
- JavaScript Functions
- Date Object
- Conditional Logic
- Form Validation
- CSS Flexbox
- Responsive Design
- CSS Class Manipulation
